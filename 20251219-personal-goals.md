# FY 2025 Personal Goals
**Role**: Architect & Team Lead | Infrastructure Team

## Objective 1: Increase Organizational Efficiency (40%)
*Build a self-sufficient team that reduces duplication and maximizes output.*

### Key Results

**1. Publish team foundation documentation** - Publish core documentation by end of FY:
   - PDR (https://www.atlassian.com/agile/product-management/requirements) outlining team scope and services
   - Technical strategy with minimum 5 ADRs/RFCs covering key architectural decisions
   - Sprint process with definition of done and escalation paths

**2. Create shared infrastructure modules** - Create shared infrastructure modules/templates adopted by at least 2 product teams by end of period

**3. Create developer documentation portal** - Create and maintain documentation portal with guides for common infrastructure tasks, reducing repeat questions and support tickets

## Objective 2: Accelerate Time-to-Market (40%)
*Provide a platform enabling product teams to release faster with minimal dependencies while maintaining operational resilience.*

### Key Results

**1. Deploy core self-service infrastructure capabilities:**
   - Kubernetes artifact promotion and deployment automation
   - Airgap environment security and isolation
   - Container vulnerability detection and scanning

**2. Document platform reliability gaps and improve incident management:**
   - Implement ticket process for infrastructure requests
   - Document current monitoring gaps and incident response challenges
   - Define P0/critical incident classification framework
   - Create FY26 roadmap for monitoring and SLA implementation

## Objective 3: Maintain Stakeholder Alignment (20%)

*Keep leadership informed through regular updates and reviews.*

### Key Results

**1. Publish regular sprint status updates:**
   - Document progress, upcoming work, and blockers each sprint
   - Share updates with Product, Engineering, and leadership teams

**2. Conduct quarterly business reviews:**
   - Present to Product, Engineering, and Executive leadership
   - Review team achievements, challenges, and upcoming priorities
